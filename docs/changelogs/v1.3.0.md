# v1.3.0

## Release date

2026-07-07

## Summary

Ships **Backup & Restore** — a Settings ▸ Backup & Restore tab that exports
14 models as an `fflate` zip (NDJSON + manifest, optional AES-256-GCM password
encryption) and re-imports them with read-only analyze, conflict
classification, and merge / replace modes (ADMIN-only). The listener dashboard
also gains a clickable event detail modal.

## Highlights

### Backup & Restore (#85, #87)

- **Export / import wizard** — zip export of 14 models with optional
  AES-256-GCM encryption; analyze → merge (skip / overwrite / rename +
  relation remap) or replace import modes; `/api/admin/backup/{export,analyze,import}` (#85)
- Allowlist the new backup routes for the docs-drift check (#87)

### Listener (#86)

- Clickable event rows open a **detail modal** (invoice, fees, preimage) (#86)
