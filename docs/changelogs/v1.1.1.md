# v1.1.1

## Release date

2026-07-06

## Summary

A patch release fixing a listener start-up crash on fresh installs: the
listener could query the `RemoteWallet` table before the web app's migrations
had created it. It now waits for the schema before subscribing.

## Highlights

### Listener (#76)

- **Fresh-install fix** — wait for web migrations before querying
  `RemoteWallet`, ending the `42P01` crash-loop on first boot (#76)
