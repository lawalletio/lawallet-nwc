# v1.1.0

## Release date

2026-07-06

## Summary

The **NWC Payment Listener** goes live. `apps/listener/` — until now an echo
stub — becomes a real transport service: it holds one NWC client per active
`RemoteWallet`, forwards NIP-47 notifications to the web app as HMAC-signed
webhooks, and reconciles its relay pool from Postgres `LISTEN`/`NOTIFY`. Around
it this release ships settings-driven listener config, hybrid missed-event
recovery, a Start9 / StartOS package, an installable PWA wallet, and the NIP-05
+ relay-picker platform polish carried over from Month 5.
