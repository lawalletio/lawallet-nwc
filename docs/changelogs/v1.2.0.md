# v1.2.0

## Release date

2026-07-06

## Summary

Listener dashboard and resilience work, plus the first admin-managed ADMIN
grants. The dashboard gains a live-count badge, activity sort, and pagination;
`/status` is hardened so it never 500s and dead LNCurl wallets auto-archive;
and the listener's HTTP API + webhook contract are now documented in the public
docs.

## Highlights

### Listener (#77, #80, #81)

- **Dashboard UX** — live-count badge, activity sort, pagination (#81)
- **`/status` hardening** — bulletproof status endpoint, keep-alive guards,
  dead-LNCurl auto-archival (#80)
- Relay status no longer misreads bare-domain relays as Disconnected (#77)

### Admin (#79)

- Admins can **grant the ADMIN role** to other users (#79)

### Docs (#78)

- Listener **HTTP API reference + webhook contract** in the public docs (#78)
